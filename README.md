# weekly-routines
Kids planner app , weekly routines

## Live Application
This application is hosted on GitHub Pages:
[https://rifaterdemsahin.github.io/weekly-routines/](https://rifaterdemsahin.github.io/weekly-routines/)

## Features
- **Visual Schedule:** Beautiful, color-coded view of Erdem family routines (Ece, Erdem, Arya, Mira).
- **Interactive Editing:** Modify meal notes or scheduled times directly on the page.
- **MongoDB Sync Option:** Hook up your own free MongoDB Data API or serverless backend for global persistence.
- **Print Optimization:** Easily print the weekly calendar directly from the browser.
